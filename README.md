# Homebrew Tap for Koe TTS

Official Homebrew tap for [Koe](https://github.com/maemreyo/zmr-koe) - Voice-controlled text-to-speech desktop app.

## Installation

```bash
# Tap this repository
brew tap maemreyo/zmr-koe

# Install Koe
brew install --cask koe
```

## Uninstall

```bash
brew uninstall --cask koe
brew untap maemreyo/zmr-koe
```

## Updating

```bash
brew update
brew upgrade --cask koe
```

## What is Koe?

**Koe** (声，Japanese for *voice*) is a macOS app that reads any selected text aloud using an on-device AI voice model.

- ✅ 100% offline & private
- ✅ Natural AI voices (Kokoro TTS)
- ✅ Global shortcuts (`⌥ Space` to read)
- ✅ ~50 MB memory footprint
- ✅ Built with Rust + Tauri

Learn more at [github.com/maemreyo/zmr-koe](https://github.com/maemreyo/zmr-koe)
